# 第6回B4輪講課題

## 概要

本課題では, 音声または音楽データを用いた分類タスクに取り組む。
ニューラルネットワークを numpy で実装し, 順伝播, 損失関数, 勾配伝播, パラメータ更新の流れを理解することを目的とする。

> [!WARNING]
> - 本課題では, ニューラルネットワークの中身を理解するため, 主要部分は numpy で実装すること
> - PyTorch, TensorFlow, scikit-learn などの機械学習ライブラリで学習処理を丸ごと置き換えないこと
> - 音声特徴量の作り方, モデル構造, ハイパーパラメータは各自で工夫してよい

## データセット `data/` について

本課題では, 次の合成音楽データを用いる。

- `data/synthetic_music.npz`

このデータには, 次の6種類の短い音が含まれる。

- soft_lead: 高調波が少ない柔らかいリード音
- bright_lead: 高調波がやや多い明るいリード音
- reed_lead: 偶数倍音も含むリード風の音
- hollow_lead: 特定の倍音が強い少し空洞感のある音
- major_chord: 長三和音風の音
- minor_chord: 短三和音風の音

各サンプルにはランダムな周波数, 音量, ノイズ, エンベロープ, 軽い残響が加えられている。
一部のクラスは音色が近く, 簡単に100%にならないようにしている。
データ生成スクリプトは解答例の参考実装として `answer/generate_data.py` に置いているが, 課題に取り組む際は配布済みの `data/` を使えばよい。

## 課題

### 6-1 音声・音楽データの観察

- `data/synthetic_music.npz` を読み込むこと。
- 各クラスの波形を可視化すること。
- フーリエ変換などを用いて, 周波数成分を可視化すること。
- 必要なら WAV に書き出して実際に音を聴き, 似ているクラス・違うクラスを確認すること。
- クラスごとにどのような違いがあるかを観察し, 簡単に考察すること。

出力画像は `fig/` に保存すること。

### 6-2 ニューラルネットワークによる分類

- 音声波形から分類に使う特徴量を作ること。
- 1層以上の隠れ層を持つニューラルネットワークを numpy で実装すること。
- ソフトマックス関数とクロスエントロピー損失を用いて多クラス分類を行うこと。
- バックプロパゲーションにより勾配を計算し, 勾配降下法で重みを更新すること。
- 学習中に以下を記録し, 可視化すること。
  - train loss
  - test loss
  - train accuracy
  - test accuracy

### 6-3 勾配伝播の確認

- 小さなデータと小さなモデルを使って, 解析的に求めた勾配と数値微分による勾配を比較すること。
- 数値微分とバックプロパゲーションの計算量の違いについて考察すること。

### 6-4 コンペ形式の性能比較

各自でモデルや特徴量を工夫し, test accuracy を比較する。

比較するとよい項目:

- 特徴量の作り方
- 隠れ層のユニット数
- 学習率
- エポック数
- 正則化の有無
- 活性化関数

発表時には, 最終的な test accuracy だけでなく, どのような工夫をしたかも説明すること。
