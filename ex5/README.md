# 第6回B4輪講課題

## 概要

本課題では, 実際に研究やベンチマークで使われる音声コマンドデータセットを用いた分類タスクに取り組む。
ニューラルネットワークを numpy で実装し, 順伝播, 損失関数, 勾配伝播, パラメータ更新の流れを理解することを目的とする。

> [!WARNING]
> - 本課題では, ニューラルネットワークの中身を理解するため, 主要部分は numpy で実装すること
> - PyTorch, TensorFlow, scikit-learn などの機械学習ライブラリで学習処理を丸ごと置き換えないこと
> - 音声特徴量の作り方, モデル構造, ハイパーパラメータは各自で工夫してよい

## データセット `data/` について

本課題では, [Speech Commands](https://www.tensorflow.org/datasets/catalog/speech_commands?hl=ja) をもとにした音声コマンド分類データを用いる。
Speech Commands は, 小さなキーワードスポッティングモデルの訓練・評価を目的に作られた, 1秒程度の英単語発話データセットである。

使用するデータ:

- `data/speech_commands_subset.npz`

このファイルは Git 管理しない。
各自で次のコマンドを実行し, Speech Commands の公式データから課題用の NumPy 形式サブセットを作成する。

```bash
uv run python answer/prepare_speech_commands.py --output data/speech_commands_subset.npz --data-dir data/speech_commands_raw
```

各サンプルは 16 kHz, 1秒の音声波形で, ラベルは次の10種類の英単語である。

- yes
- no
- up
- down
- left
- right
- on
- off
- stop
- go

公式データセット全体は train 85,511件, validation 10,102件, test 4,890件を持つが, 全体をそのまま扱うと大きいため, 本課題では扱いやすいサイズのサブセットを作成する。
初回実行時は公式 tar.gz のダウンロードに時間がかかる。
生成される `data/` と `answer/data/` は `.gitignore` で除外している。

## 課題

### 6-1 音声コマンドデータの観察

- `data/speech_commands_subset.npz` を読み込むこと。
- 各クラスの波形を可視化すること。
- フーリエ変換などを用いて, 周波数成分を可視化すること。
- 必要なら WAV に書き出して実際に音を聴き, 似ているクラス・違うクラスを確認すること。
- 単語ごとにどのような違いがあるか, また話者差・発音差・ノイズがどのように影響しそうかを考察すること。

出力画像は `fig/` に保存すること。

### 6-2 ニューラルネットワークによる分類

- 音声波形から分類に使う特徴量を作ること。
- 1層以上の隠れ層を持つニューラルネットワークを numpy で実装すること。
- ソフトマックス関数とクロスエントロピー損失を用いて多クラス分類を行うこと。
- バックプロパゲーションにより勾配を計算し, 勾配降下法で重みを更新すること。
- 学習中に以下を記録し, 可視化すること。
  - train loss
  - test loss
  - train accuracy
  - test accuracy

### 6-3 勾配伝播の確認

- 小さなデータと小さなモデルを使って, 解析的に求めた勾配と数値微分による勾配を比較すること。
- 数値微分とバックプロパゲーションの計算量の違いについて考察すること。

### 6-4 コンペ形式の性能比較

各自でモデルや特徴量を工夫し, test accuracy を比較する。

比較するとよい項目:

- 特徴量の作り方
- 隠れ層のユニット数
- 学習率
- エポック数
- 正則化の有無
- 活性化関数
- validation split を使ったハイパーパラメータ選択

発表時には, 最終的な test accuracy だけでなく, どのような工夫をしたかも説明すること。
